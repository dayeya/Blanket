#                ===== Picky =====          
Picky is an open-source web application firewall project.

* Picky acts as a reverse proxy between the host and the web server.

                     +-------------+
   +------+          |    Web      |          +------------+
   | Host | => => => | Application | => => => | Web server |
   +------+          |  Firewall   |          +------------+
                     +-------------+
